# KResume (AI compact)

Developer-First Resume IDE. Desktop app (Compose Multiplatform). Still building.

## PoC: Runtime Compilation Server

Local web server (`:server` module) with browser IDE — type DSL, see
live HTML preview. Validates compilation speed for hot-reload UX.

**Run:** `./gradlew :server:run` → http://localhost:8080

**Engines:** `--engine=raw` (default), `jsr223`, `kctfork`

**Endpoints:** `POST /api/compile` | `/api/bench` | `/api/export` (PDF)

**Frontend:** CodeMirror 5 editor + iframe preview, 200ms debounce.
Benchmark page at `/bench` with side-by-side chart.

**PDF:** OpenHTMLtoPDF, table-based layout, ATS-friendly.

**ATS:** Semantic HTML, position/company separate, contact visible.
See `docs/ats-compatibility.md` for checks.

## Features (Goal)

- Kotlin DSL for resume data
- Live preview
- Import/Export: multiple formats
- Private & offline
- Optional AI assistant

## Tech

Kotlin, Compose Multiplatform, Ktor 3.5.1 (PoC)

## Build

```bash
./gradlew build
./gradlew test
./gradlew :server:run    # start PoC
```
